时频分布的基本性质要求

1. 实质性

时频分布必须是实数，且是非负的。因为能量必须是实数而不能是负数，同时理想情况下，能量也不能是负数。

2. 能量守恒

3. 边缘特性

对频率 $f$ 积分能得到瞬时功率 $|x(t)|^2$

对时间 $t$ 积分能得到能量谱密度 $|X(f)|^2$

4. 一阶矩

瞬时功率 $f_i(t) = \frac{\int_{-\infty}^\infty f \cdot P(t, f)df}{\int_{-\infty}^\infty P(t, f)df}$，表示在某一时刻 $t$ 信号的平均频率是多少

群延迟 $\tau_g(f) = \frac{\int_{-\infty}^\infty t \cdot P(t, f)dt}{\int_{-\infty}^\infty P(t, f)dt}$，表示在某一频率 $f$，信号的能量中心大概出现在什么时间

5. 有限支撑性

如果信号在某个时间或频率外是 0，则此处分布也该是 0。强有限支撑指只要边缘分布为 0 的地方，则联合分布也必须为 0

基本性质

  1. 频移不变

如果原信号在时域乘以一个复指数信号（即进行了频率调制，频率搬移了 $f_0$），那么其 STFT 的结果在频率轴上也会相应地平移 $f_0$。

  1. 时移变

如果原信号在时间上延迟了 $t_0$，其 STFT 的结果在时间轴上也会延迟 $t_0$，但同时会引入一个相位因子 $e^{-j2\pi f t_0}$。因此不具有时移不变性

离散化

表示在第 $m$ 个时间帧，用频率为 $nF$ 的复指数对加窗后的信号进行调制和求和。

逆变换：

通过叠加无数个“时频原子”（即被平移和调制的综合窗函数 $g$），利用 STFT 系数作为权重，来还原原始信号 $z(k)$。

在离散 STFT 中，为了消除由采样引起的混叠误差，必须满足非常严格的数学关系。

Cohen 变量的四种类型

基于你提供的笔记图片内容以及Cohen类的完整理论体系，Cohen类分布主要包含以下四种类型。

以下是针对这四种类型及其相互关系的言简意赅的回答：

  1. 能量化 Cohen 类（$C_E$，移不变类

    • 核心特征：具有时移和频移不变性。
    • 定义：时频分布 $P(t, f)$ 可通过时频卷积由 Wigner-Ville (WV) 分布导出。
    • 表达式：(即：核函数 $\Phi$ 与 WV 分布的卷积)

  2. 相关化 Cohen 类（$C_C$，对偶类）

    • 核心特征：是能量化 Cohen 类的对偶形式，与模糊函数密切相关。
    • 定义：在模糊域（$\tau, v$）中，分布表示为核函数与模糊函数的乘积。
    • 表达式：(即：核函数 $\phi$ 乘以 模糊函数 $A_z$)

  3. 仿射类（$A_E$，尺度不变类）

    • 核心特征：能够保持时间尺度不变（缩放特性）和时移不变。
    • 定义：如果信号在时间上进行了缩放（如 $t \to at$），其时频分布也进行相应的仿射变换。
    • 表达式：任何仿射类分布 $P_z$ 都可以表示为 WV 分布的仿射变换（核函数 $X$ 与信号无关）。

  4. 交集类（$CA_E$，移位-尺度不变类）

    • 核心特征：同时具备移位不变性（$C_E$）和尺度不变性（$A_E$）。
    • 定义：其核函数仅取决于 $\tau$ 和 $v$ 的乘积（即乘积核）。

它们之间的关系

  1. 卷积与乘积的对偶关系（$C_E$ 与 $C_C$）：

    • 能量化类（$C_E$）和相关化类（$C_C$）互为二维傅里叶变换关系。
    • 计算复杂度的转换：在时频域中复杂的卷积运算（$C_E$），在模糊域中转化为简单的乘积运算（$C_C$）。这使得在模糊域设计滤波器（核函数）更加容易。

  2. 移位与缩放的互补关系（$C_E$ 与 $A_E$）：

    • $C_E$ 保证了信号平移时分布跟着平移（适合稳态分析）。
    • $A_E$ 保证了信号缩放时分布跟着缩放（适合多分辨率/小波分析）。

  3. 核心联系：

    • 所有的 Cohen 类分布本质上都是对 Wigner-Ville 分布 (WV) 的某种变换（卷积、乘积或仿射变换）。WV 分布是它们的“母体”或基础。

这部分探讨了一个理论上的悖论：

  • 问题：对于单分量 LFM 信号，我们实际上取了 $\Phi(\tau, v) = 1$（即没有加窗，直接用 WVD）。但是 LFM 信号明显是非平稳信号（频率随时间变化），为什么可以使用“无穷宽的窗”（即不加窗）呢？通常非平稳信号需要加窗来截断以保证短时平稳。
  • 解释：笔记指出，因为 LFM 信号的二次型是平稳的（”因为其二次型为平稳的”）。
    • 解读：这里的“二次型”指的是 WVD 定义中的瞬时相关函数 $z(t+\tau/2)z^*(t-\tau/2)$。对于 LFM，这个乘积关于 $\tau$ 是一个纯粹的复正弦波（等幅，频率固定为瞬时频率），在积分变量 $\tau$ 的维度上，它表现得像一个平稳的正弦信号。
  • 结论：因此，对于 LFM，可以采用 $\Phi(\tau, v)=1$（即不加窗/不滤波）。
  • 引申：笔记最后一句指出，“当非平稳信号的二次型不满足平稳时，应加适当的窗。”这意味着对于比 LFM 更复杂的信号（例如非线性调频），WVD 可能就不再是完美的线，这时就需要引入 Cohen 类的其他核函数（加窗）来改善性能。

总结，这张笔记主要证明了 Wigner-Ville 分布 (WVD) 对于 线性调频信号 (LFM) 拥有最完美的时频聚集性，其分布完全集中在瞬时频率轨迹上。这是 WVD 的一大优势，但同时也引出了后续关于如何处理更复杂非平稳信号的讨论。

交叉项抑制

交叉项是二次型时频分布（如Wigner-Ville分布）的主要缺陷，会产生虚假的信号特征。

  • 前文可能只分析了简单的双分量单频信号，但在一般的多分量信号中，交叉项的分布和影响要复杂得多。
  • 核心手段：抑制交叉项的主要手段是核函数的设计。

有限支撑性的要求

  • 为了保证时频分布的物理意义合理（即信号在某段时间或频段为0，其分布也应为0），要求核函数满足“弱有限支撑性”。
  • 数学条件：
    • 时域约束：$\psi(t, \tau) = 0, \quad |t| > |\tau|/2$
    • 频域约束：$\Psi(f, v) = 0, \quad |f| > |v|/2$

  • 这两个条件保证了分布具有有限的时间和频率支撑。

  • 笔记指出，想要完全消除交叉项是不可能的，除非让核函数 $\Psi(f, v) \equiv 0$，但这会导致信号项也没了。

  • 特定位置的抑制：如果我们希望交叉项不要出现在信号频率之外的地方（即不要凭空产生虚假频率），需要满足更严格的条件：
    • $\Psi(f, v) = 0, \quad \forall |f| \neq |v|/2$
    • $\psi(t, \tau) = 0, \quad \forall |t| \neq |\tau|/2$
  • 满足这些条件可以保证：在信号 $z(t)=0$ 的时间段内，交叉项不会出现；在信号频谱为0的区域，交叉项也不会出现。
  • 局限性：这些条件只能保证“无中生有”的交叉项不出现，但无法消除信号分量之间的交叉项。

核心策略：二维低通滤波

  • 这是Cohen类设计核函数最核心的指导思想。
  • 原理：
    • 信号项（自项）：一般位于模糊平面（Ambiguity Plane，即 $\tau-v$ 平面）的原点附近。
    • 交叉项：通常远离原点（因为交叉项对应的是两个分量时频中心的差值）。
  • 结论：因此，希望核函数 $\phi(\tau, v)$ 是一个二维低通函数。
    • 它相当于一个掩膜（Mask），保留原点附近的信号项，滤除远离原点的交叉项。

矛盾与权衡

  • 笔记强调了一个重要的矛盾：交叉项的减小与信号项的维持是一对矛盾。
  • 因为低通滤波在滤除交叉项的同时，不可避免地会模糊信号项（降低时频分辨率）。

权衡的代价

  • 接上一页，减小交叉项必然会对信号项产生“拉平”或“模糊”的副作用（笔记中称为“副作用”）。
  • 无解情况：如果信号项和交叉项在模糊域中发生重叠，目前还没有方法能只减小交叉项而不损伤信号。因此，通常假设两者无重叠。

除了上述的“核函数设计法”外，笔记还介绍了三种改善 Cohen 类时频分布性能的方法：

  1. 信号分解法（Signal Decomposition）：

    • 将解析信号分解为若干基本分量。
    • 分别计算各分量的时频分布，然后求和。
    • 优点：如果分解得当，各分量独立计算，自然就没有了分量间的交叉项。

  2. 图像处理法：

    • 利用交叉项和信号项在几何形状和振荡结构上的不同。
    • 交叉项通常具有震荡特性，可以使用图像处理算法试图将其滤除。

  3. 重排法（Reassignment Method）：

    • 通过对信号进行重排（Reallocation / Reassignment）。
    • 将分布的能量从计算位置移动到重心位置，从而极大提高信号分量的时频聚集性，使谱图更加清晰。

这两页笔记系统地阐述了 Cohen 类时频分布中“交叉项抑制”的方法论：

  • 根本原理利用模糊域中信号项（低频/原点）和交叉项（高频/远处）的分离特性。
  • 实现方式：设计呈低通特性的核函数（如 Choi-Williams 分布等）。
  • 代价：牺牲一定的时频分辨率。
  • 扩展：当核函数设计无法满足需求时，可以采用信号分解或重排等高级技术。

  • 与 Classen 定义的对比（打星号 $\star$ 部分 - 重点）：

    1. 频率边缘特性恢复：解释：对时间求和精确等于信号的功率谱，消除了 $\pi$ 处的混叠项。这是该定义最大的优势。
    2. 时间边缘特性：解释：对频率积分可以得到瞬时功率，但只在偶数点有效。

总结

这两页笔记主要讲述了离散 WVD 的困境：

  • 直接离散化（Classen定义）会导致频谱周期变为 $\pi$，从而产生混叠，无法完美保持连续时间 WVD 的性质（如边缘分布）。
  • 改进的定义（Peyrin定义）虽然解决了频谱混叠问题（满足频率边缘特性），但在时域上引入了新的限制（偶数点特性）。
  • 这印证了开头所说的“至今尚无公认的完美定义”，不同的定义适用于不同的应用场景。

第二题：双分量高斯信号的 WVD 与模糊函数

题目：
非平稳信号 $z(t) = x_1(t) + x_2(t)$，其中 $x_1, x_2$ 为高斯信号，中心分别为 $(t_1, w_1)$ 和 $(t_2, w_2)$。求 $z(t)$ 的 WV 分布及模糊函数，画草图，并分析信号项与交叉项。

【解答步骤】

1. Wigner-Ville 分布 (WVD)

根据 WVD 的双线性性质：

  • 信号项（自项）：
    由第一题结论可知，这是两个分别位于 $(t_1, w_1)$ 和 $(t_2, w_2)$ 的高斯“山峰”。

  • 交叉项（互项）：
    交叉项位于两分量的几何中心，即 $(\frac{t_1+t_2}{2}, \frac{w_1+w_2}{2})$ 处。
    经过推导（略去繁琐积分过程），交叉项具有如下形式：

    • 特征：这是一个位于两信号中心连线中点的高斯包络，内部被余弦波调制（震荡）。震荡的频率取决于两分量在时频平面上的距离。

WVD 图像草图描述：

  • 两个正峰：分别在 $(t_1, w_1)$ 和 $(t_2, w_2)$，代表真实的信号能量。
  • 一个震荡项：位于两峰连线的中点。它的幅度可能比信号项还大（是信号项的2倍），且正负交替震荡。这是一个虚假信号（Artifact），会干扰对信号的判读。

2. 模糊函数 (Ambiguity Function, AF)

模糊函数定义为 $A_z(\tau, v) = \int z(t+\frac{\tau}{2})z^*(t-\frac{\tau}{2}) e^{-j v t} dt$。
同样利用叠加性：

  • 信号项（自项）：
    模糊函数的自项总是位于模糊平面的原点 $(0, 0)$。

    这是两个叠加在原点的高斯凸起。

  • 交叉项（互项）：
    交叉项位于两分量参数差值的位置。

    分布在 $(\pm(t_1-t_2), \pm(w_1-w_2))$ 处。

    也就是说，交叉项远离原点，出现在 $(\Delta t, \Delta w)$ 和 $(-\Delta t, -\Delta w)$ 处。

模糊函数图像草图描述：

  • 中心主峰：位于原点 $(0,0)$，幅度最大，由两个自项叠加而成。
  • 两个侧峰：位于原点两侧对称位置，距离原点越远说明两个信号分量在时频面上分得越开。

3. 信号项与交叉项的分析

比较维度 Wigner-Ville 分布 (WVD) 模糊函数 (AF)
信号项位置 位于真实的 $(t, \omega)$ 坐标处。 位于模糊域原点 $(0,0)$ 附近。
交叉项位置 位于两个信号项的中间。 位于远离原点的 $(\Delta \tau, \Delta v)$ 处。
交叉项特征 剧烈震荡，破坏信号的可读性。 平滑的高斯包络，与信号项在空间上分离。
分析 WVD 的交叉项是其最大的缺陷，容易让人误以为中间存在第三个分量。 AF 通过将交叉项从原点移开，为通过滤波（加窗核函数）消除交叉项提供了可能（即 Cohen 类的基本思想）。

总结：
WVD 直观展示了信号的时频分布，但受交叉项干扰严重；模糊函数虽然不直接显示时频分布，但它成功地将信号项（聚集在原点）和交叉项（分散在四周）分离开来，这正是 Cohen 类时频分布设计核函数以抑制交叉项的理论基础。
